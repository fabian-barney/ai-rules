# UPDATE

Instructions for AI agents to update ai-rules in a downstream repository.

## Prompt Examples
- "update ai-rules to version v2.0.0"
- "update ai-rules to the latest tagged release"

## Expectations
- Prefer a tagged release unless explicitly asked for a branch.
- Preserve local customizations outside the ai-rules subtree or vendor path.
- Record the updated version in the destination repository if it tracks versions.
